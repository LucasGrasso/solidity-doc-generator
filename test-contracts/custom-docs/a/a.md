# Getting Started with Test Contracts

This guide will walk you through understanding and using the test contracts in this project.

## What are the Test Contracts?

The test contracts are sample Solidity smart contracts used for documentation generation testing. They demonstrate various contract patterns and features.

## Viewing the API Documentation

This site automatically generates API documentation for all contracts. You'll find:

- **Contract specifications** — Method signatures and state variables
- **Function documentation** — What each function does and its parameters
- **Event definitions** — Events emitted by the contracts
- **Error types** — Custom errors and their messages

## Running Tests

To run tests against the contracts:

```bash
npm test
```

## Compiling Contracts

To compile the Solidity contracts:

```bash
npx hardhat compile
```

## Next Steps
